This tool is a JSON data saving node designed specifically for Unity Visual Scripting users.
Although it is powered by C# behind the scenes, it was created with non-programmers in mind,
allowing users who are not familiar with code to easily save and manage data using Visual Scripting alone.

:: Key Features ::

Save various GameObject variables such as:
Integers
Floats
Strings
GameObjects
And more

Store multiple objects in a single JSON file
Assign and manage data using unique IDs for each object
This node is designed to be simple, flexible, and practical for visual workflows,
making data persistence accessible without writing custom code.
